Voc um analista de dados de uma loja on-line e recebeu uma base de dados contendo informaes de vendas. Seu objetivo analisar os dados para entender o perfil dos clientes e o desempenho das vendas. A base de dados contm as seguintes colunas: Nome, Idade, Cidade, Vendas.

Tarefa: Utilize os conhecimentos adquiridos sobre pandas e numpy para o passo a passo a seguir:

1.Crie um DataFrame com base nos dados fornecidos;

2.Filtre os clientes que tm mais de 30 anos;

3.Calcule a mdia de idade dos clientes;

4.Substitua valores faltantes na coluna Cidade por Desconhecido;

5.Calcule a soma total das vendas.

Dados de Entrada:

dados = {

'Nome': ['Ana', 'Bruno', 'Carlos', 'Daniela', 'Eduardo'],
